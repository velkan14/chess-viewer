# Chess Viewer

React.js application to view Chess Boards and load Fen.

## Getting Started

### Prerequisites

Make sure you have installed NodeJS.

```bash
$ node -v
v14.16.0
```

### Installing

```bash
https://github.com/velkan14/chess-viewer
cd chess-viewer
npm install
npm start
```
